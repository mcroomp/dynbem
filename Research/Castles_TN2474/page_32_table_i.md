### Flagged items

**A — Run 3 row order (RESOLVED):**
Source image had last three rows in non-ascending order (V/ΩR = 0.0799, 0.0762, 0.0795).
User has re-verified source and confirmed current order 0.0762 / 0.0795 / 0.0799 is correct. ✓

**B — Run 9 hover λ₁(Q) = 1.41 (probable misread):**
At hover all other runs with λ₁(Q) present give 1.06–1.16. A value of 1.41 at λ₂=0 is
anomalously high. "1.41" could be a transposition of "1.14", consistent with λ₁(T)=1.05.
Flag for re-verification against source image.

**C — Run 34 duplicate λ₁(T) = 1.28 at λ₂ = 2.22 and 2.39:**
At V/ΩR=0.0624 and V/ΩR=0.0690 both rows record λ₁(T)=1.28. In deep descent λ₁ should
decrease monotonically as λ₂ increases. Expected λ₁(T) at λ₂=2.39 ≈ 1.20–1.25.
Verify second row against source.

**D — Run 36 hover ΔCQ anomalously low:**
Hover ΔCQ = 0.000242 versus 0.000282–0.000292 in Runs 3/4/5 (same CT=0.004, 1200 rpm) —
14–17% below peers. Run 36 hover collective θ=8.49° is ~3–4% below Runs 3/4/5 (8.66–8.86°),
which may explain part of the gap, but not all. Verify source; also possible torque gauge
offset shifted between runs.

**E — Run 38 ΔCQ missing for 14 of 20 rows:**
ΔCQ is blank from row 1 through V/ΩR=0.0765 (all hover and VRS points), then present for the
last six WBS rows. Torque transducer likely out of range or not connected during the slow-
descent phase of this run. λ₁(T) column appears complete and is the primary data here;
Run 14 (same CT=0.005, 1600 rpm) provides ΔCQ reference for the VRS/hover regime.

**F — Run 32 ΔCQ non-monotonic at deep descent:**
V/ΩR=0.0721 (λ₂=2.21): ΔCQ = −0.000092.
V/ΩR=0.0771 (λ₂=2.37): ΔCQ = −0.000069 (less negative, i.e. reduced autorotative torque).
Deeper descent should yield more energy recovery (more negative ΔCQ). Both values are small
(within ±0.0001 of zero) so measurement noise may account for the reversal. Run 9 (CT=0.002,
1600 rpm) shows smooth monotonic decrease through the same λ₂ range.

**G — Run 36 large λ₁(T)/λ₁(Q) divergence in VRS entry:**
Near VRS entry (V/ΩR = 0.0459–0.0715), λ₁(Q) exceeds λ₁(T) by 0.17–0.69:
