## Rotor geometry

- 2 blades, NACA 0012, untwisted, untapered
- Radius R = 1.143 m (4.5 ft / 54 in), chord c = 0.1905 m (7.5 in)
- Solidity ╧Г = Nc/(╧АR) = 0.1062
- Test conditions: ╬╕c = 0┬░, 2┬░, 5┬░, 7┬░, 8┬░, 10┬░, 12┬░ collective
- RPM range: 650 тАУ 2540 rpm (M_tip = 0.230 тАУ 0.894)

## What this paper does NOT contain

- **No CP / torque / power data** тАФ this is a blade-pressure + wake-geometry study, not a
  performance (thrust-torque) test. There is no table of CP values anywhere.
- No inflow velocity measurements.
- No off-axis or forward-flight data (hover only).

For CP-CT polar validation, a different dataset is needed, e.g.:
- Harrington (1951) NACA-TN-2165 тАФ clean hover CT vs CP for two rotors
- Sikorsky UTTAS or similar publicly available rotor performance data
